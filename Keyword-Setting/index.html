📦 키워드 → JSON 변환기

키워드 데이터를 입력하면 키워드 대화 위젯 생성기용 JSON으로 변환합니다 (시퀀스 대사 지원).

📝 텍스트 형식 (분류별)
📊 표 형식 (| 구분)
※ `[분류명]` 줄로 시작, `키워드: 설명` 형식. 대사/표정은 기본 시퀀스(한 줄)로 생성됩니다.
📌 대사 형식:
💡 표 형식 규칙 (열 수에 따라 다름)
- 단순 대사 모드: 3열(기본) / 4열(대사 JSON) / 5열(대사+표정)
- 시퀀스 대사 모드: 3열(기본: `[["예시 대사"]]`) / 4열(시퀀스 대사 JSON) / 5열(시퀀스 대사+표정 JSON)
- JSON은 반드시 유효한 중첩 배열 형식이어야 합니다: [["첫째","둘째"], ["셋째"]]
- 표정도 동일한 중첩 구조: [["기본","웃음"], ["정색"]] 
// 결과가 여기에 표시됩니다
💡 시퀀스 대사란? 키워드 하나가 여러 개의 독립적인 대화 흐름(시퀀스)을 가지며, 클릭 시 무작위로 선택됩니다.
선택된 시퀀스 내에서는 대사가 순서대로 출력되고, "다음" 버튼으로 넘어갑니다.
예: [["대사1-1","대사1-2"], ["대사2"]] → 50% 확률로 2줄 대화, 50% 확률로 1줄 대화.
