📦 마비노기 키워드 → JSON 변환기

키워드 데이터를 입력하면 키워드 대화 위젯 생성기용 JSON으로 변환합니다.

📝 텍스트 형식 (분류별)
📊 표 형식 (| 구분)
※ `[분류명]` 줄로 시작, `키워드: 설명` 형식. 대사/표정은 기본값으로 생성됩니다.
📌 포함할 정보:
💡 표 형식 규칙
- 구분자는 `|` (파이프)입니다.
- 열 순서: 분류 | 키워드 | 설명 [ | 대사(JSON) [ | 표정(JSON) ] ]
- 대사/표정은 반드시 유효한 JSON 배열 형식이어야 합니다.
- 대사 없이 표정만은 불가능합니다 (대사 포함 시 표정 선택 가능). 
// 결과가 여기에 표시됩니다
💡 사용 방법 요약
- 텍스트 형식: 빠른 키워드+설명 입력 (대사/표정 기본값)
- 표 형식: 열 개수에 따라 자동 감지 (3열: 기본, 4열: 대사 포함, 5열: 대사+표정)
- 대사/표정은 반드시 JSON 배열 예: ["첫 대사","둘째 대사"]
- 생성된 JSON은 키워드 대화 위젯 생성기의 categories 배열에 그대로 사용 가능합니다.
